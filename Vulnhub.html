Vulnhub

Kioptrix
Level 1

Kioptrix was a

Lord of the root

This box was based upon some funamentals in penetration testing, including web exploitation via a blind time based SQL injection and a buffer overflow in an insecure binary to privilege escalate to root, bypassing ASLR.

Prime 1

This stressed the need for goof web enumeration / fuzzing skills to get a foothold. Once on the network there were a series of challenges that involved decrypting messages and a type of path poisining attack to get root.
