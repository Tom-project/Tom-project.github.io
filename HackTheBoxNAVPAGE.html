Hack The Box

Academy

Academy was a simple box that stressed the need for good grepping skills. It involved a vulnerable php web manager "Lavarel". Finding creds for two different accounts and then exploit a vulnerable binary to get root.

Doctor

This was a nice and simple box with a few rabbit holes. It involved a template injection vulnerability, leading to a foothold on the network, and then an authenticated RCE for privilege escalation via the splunk management port on 8089.

Omni

Omni was an IOT box, involving an RCE on a Windows IOT core device and an interesting powershell technique to view secure string passwords allowing us to get administrator privileges and reading the root flag.

Traceback

Traceback was a linux machine which involved a website that had already been owned with a backdoor on the webserver, with a two step process to root, involving a luvit binary exploit and injecting malicious code into the ssh banner for root privilege escalation.

Buff

Buff was running a vulnerable webserver, hosting "Gym management software 1.0", allowing for unauthenticated RCE and a buffer overflow to root.
